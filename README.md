# Тестовое задание 1

Это первое тестовое задание. Посмотреть финальную версию можно [здесь](https://nosdmitry.github.io/header-task).

Второе тестовой задание можно посмотреть по [ссылке](https://github.com/nosdmitry/map-api).

----

### Ключевые моменты:

* BEM
* Grid/Flex layout
* Webpack
* Sass
* Вёрстка полностью адаптивна к экранам с любыми разрешениями и к смартфонам

### Установка:

- Скачать проект 
- Команда `npm run start` для разработки
- Команда `npm run build` для финальной сборки
